## Main Features

- **Batch processing** — drag and drop folders or individual files and process them all at once.
- **Format conversion** — JPEG, PNG, WebP, AVIF, GIF, TIFF, HEIC, BMP. Per-format codec options.
- **Quality control** — compress with adjustable quality settings.
- **Resize** — Fit, Contain, Fill, and Stretch modes. Smart Crop that automatically focuses on the most important part of the image. Configurable resampling (Lanczos3, Mitchell, Nearest Neighbor).
- **Transforms** — rotate, mirror, trim transparency, extend canvas, flatten transparency to background color.
- **Effects & Adjustments** — 24 effects with multi-instance support and drag-and-drop reordering:
  - _Tone:_ Exposure, Brightness/Contrast, Shadows/Highlights, Curves, Levels, Temperature, Normalize
  - _Color:_ Hue/Saturation, Vibrance, Color Balance, Color Grading, Color Mixer, Channel Mixer, Grayscale
  - _Detail:_ Sharpen, Blur, Noise Reduction, Noise/Grain
  - _Creative:_ Posterize, Threshold, Invert, Vignette, Tint, Gradient Map
- **Preview** — live Before/After comparison. Lightbox with filmstrip navigation and collapsible settings.
- **Presets** — save and load your settings, import/export preset files.
- **Output options** — save alongside originals, overwrite originals, or output to a separate folder with option to keep the original folder structure. Configurable suffix. Optional metadata stripping.
