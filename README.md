# Bytesize Pic

A free offline batch image optimizer for Windows. No uploads, no file limits, no account required. Everything runs locally.

## Main Features
- Batch process entire folder structures at once
- Convert between formats (JPEG, PNG, WebP, AVIF, TIFF, GIF, HEIC, HEIF)
- Compress with adjustable quality settings
- Color tinting and invert
- Save and load presets
- Grid view with live previews before export
- Right-click context menu on files and folders

## Requirements
- Windows 10 or later
- ~112MB disk space

## Installation
Download and run the .exe installer. Windows may show a SmartScreen warning since the app isn't code-signed yet. Click "More info" then "Run anyway."
